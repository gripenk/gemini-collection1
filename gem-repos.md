# Gemini Gems & Agent Skills: GitHub Repository Collection

This document serves as a curated directory of GitHub repositories dedicated to collecting "Gems" (system instructions, personas, and agent skills) for Google Gemini. 

These repositories contain raw Markdown prompts that can be copied directly into a Gem's instructions or linked as a knowledge base.

---

## 1. OpenDataHub AI Helpers
**Repository:** `opendatahub-io/ai-helpers`
**Focus:** Developer tools, technical execution, and open-source AI integrations.

### Description
A collaborative marketplace for AI tools with a dedicated `helpers/gems` directory. This repo is built for users who want to turn their Gemini into a rigorous technical assistant.

### What You'll Find
*   Code review and refactoring personas.
*   System instructions for generating technical documentation.
*   Gems designed to mimic other AI environments (like Cursor or Claude Code) for cross-compatibility.

---

## 2. Awesome Gemini AI
**Repository:** `ZeroLu/awesome-gemini-ai`
**Focus:** Comprehensive curation, prompt engineering, and diverse use-cases.

### Description
Part of the famous "Awesome" GitHub lists, this is one of the most extensive collections of high-performance Gemini prompts sourced from prompt engineers across Reddit and X.

### What You'll Find
*   Massive, highly detailed system prompts.
*   Categories spanning coding, design, creative writing, and productivity.
*   Autonomous AI agent instructions (prompts that instruct Gemini on how to break down complex tasks into sequential steps).

---

## 3. Oharu121's Skills & Gems
**Repository:** `oharu121/skills`
**Focus:** Niche functionalities, workflow automation, and specific daily tasks.

### Description
A specialized repository containing custom "skills" for various AI agents, featuring a distinct section built entirely for Gemini Gem system prompts.

### What You'll Find
*   **Daily-Briefing Gem:** Instructions designed to synthesize schedules, emails, and daily goals.
*   **Thumbnail-Generator Gem:** Prompts optimized to help Gemini conceptualize and generate visual metaphors for articles and presentations.
*   Highly specific formatting guardrails for predictable AI outputs.

---

## 💡 How to Use This Document with a Gem

1. Save this file as `gem-repos.md` in your own GitHub repository.
2. In the Google Gemini Gem Manager, paste the RAW URL of this file into the Knowledge section.
3. Use the following Instruction in your Gem:
   > *"You are a Prompt Librarian. Read the attached `gem-repos.md` file. When I ask for a specific type of capability or prompt, recommend which of these three repositories I should search, and explain why based on their Focus and Description."*
